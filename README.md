# คำนวณไทยช่วยไทย พลัส 60/40

Landing Page ภาษาไทยสำหรับคำนวณยอดใช้สิทธิไทยช่วยไทย พลัส 60/40 จากสิทธิรัฐคงเหลือ โดยเว็บนี้เป็นเครื่องมือคำนวณอย่างไม่เป็นทางการ และไม่เกี่ยวข้องกับหน่วยงานรัฐ

## ฟีเจอร์

- กรอกสิทธิรัฐคงเหลือแล้วคำนวณผลทันที
- แสดงรัฐช่วยได้จริง ยอดซื้อสูงสุด และเงินที่ประชาชนต้องจ่ายเอง
- คำนวณตามสูตรรัฐช่วย 60% ประชาชนจ่าย 40%
- จำกัดสิทธิรัฐที่ใช้ได้ต่อวันไว้ที่ 200 บาท
- รองรับมือถือและ Desktop
- มี SEO meta, JSON-LD, robots.txt และ sitemap.xml
- เตรียมตำแหน่ง Ad Placeholder สำหรับ Google AdSense ในอนาคต

## สูตรคำนวณ

```js
สิทธิรัฐที่ใช้ได้จริง = Math.min(สิทธิรัฐคงเหลือ, สิทธิที่ใช้ได้ต่อวัน)
ยอดซื้อสูงสุด = สิทธิรัฐที่ใช้ได้จริง / 0.60
เงินที่ประชาชนต้องจ่าย = ยอดซื้อสูงสุด * 0.40
```

ตัวอย่าง: ถ้าสิทธิรัฐคงเหลือ 200 บาท และสิทธิที่ใช้ได้ต่อวัน 200 บาท

- รัฐช่วยได้จริง: 200.00 บาท
- ซื้อของได้สูงสุด: 333.33 บาท
- ประชาชนจ่ายเอง: 133.33 บาท

## วิธีติดตั้งและรัน

ติดตั้ง dependencies:

```bash
npm install
```

รันเว็บสำหรับพัฒนา:

```bash
npm run dev
```

Build สำหรับ production:

```bash
npm run build
```

Preview ไฟล์ production build:

```bash
npm run preview
```

## GitHub Pages

โปรเจกต์นี้ตั้งค่า GitHub Pages ผ่าน GitHub Actions แล้วใน `.github/workflows/deploy.yml`

เมื่อต้องการ deploy:

1. Push ไปที่ branch `main`
2. ไปที่ GitHub repository
3. เปิด `Settings` -> `Pages`
4. ตั้ง `Build and deployment` เป็น `GitHub Actions`
5. ตรวจผล deploy ที่แท็บ `Actions`

URL ที่ตั้งไว้สำหรับ GitHub Pages:

```text
https://ooy30.github.io/thai-help-calculator/
```

หมายเหตุ: `vite.config.js` ตั้ง `base` เป็น `/thai-help-calculator/` เพื่อให้ asset path ทำงานถูกต้องบน GitHub Pages แบบ repository site
